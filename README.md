## Запуск приложения

Для запуска приложения необходим запущенный docker\
Выполните эту команду в терминале в директории с приложением

```bash
$ docker-compose up -d
```

## Unit test

Для запуска тестов необходимо ввести команду

```bash
$ npm run test
```
